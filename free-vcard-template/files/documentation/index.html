↓   How can I add my own images to about section?

Just name the images that you want to use in about section image1.png, image2.jpg and image3.jpg and replace them with the existing ones under style/images/ folder. To change the thumbnails, first you should prepare thumbnails with dimensions of 40x29px and name them as ath1.png, ath2,jpg and ath3.jpg respectively.

The big images must be 134x134px.

↓   How can I add an image to portfolio?

You can have any number of images as you like. There are 4 images in a row.

The image 1.jpg is the fullsize image name and th1.jpg is the thumbnail image.

↓   How can I receive messages sent via Contact Form?

To receive the messages sent through the contact form, open up contact/form-handler.php and edit line 5:

$emailto = 'yourname@yourdomain.com';

For SMTP support edit line 4. Set to "1" if you want to use SMTP to send the email. Default is "0", which uses the standard PHP mail function.

$use_smtp = '1';

If using SMTP you will also need to fill in the 4 SMTP variables - lines 64 - 67.

$SmtpServer = 'SMTP SERVER';
$SmtpPort = 'SMTP PORT';
$SmtpUser = 'SMTP USER';
$SmtpPass = 'SMTP PASSWORD';
